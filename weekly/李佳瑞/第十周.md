2. 修复 TypeScript “对象可能为未定义”
在你的 index.vue 脚本中，TS 报错是因为它无法百分之百保证 groups[r.name] 已经是一个数组。

推荐修改方案：
将第 204-207 行的代码改为下面这种更现代、更安全的写法：

TypeScript
filtered.forEach(r => {
  // 如果 groups[r.name] 不存在，则初始化为一个空数组
  // 使用逻辑空赋值符号 ||= 让代码更简洁
  (groups[r.name] ||= []).push(r);
});
或者使用更直观的判断：

TypeScript
filtered.forEach(r => {
  if (!groups[r.name]) {
    groups[r.name] = [];
  }
  // 使用 ! 告诉 TS 你确定它现在不是 undefined
  groups[r.name]!.push(r);
});
